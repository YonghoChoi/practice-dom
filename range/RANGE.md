# Range

* 개요
  * 콘텐츠 범위를 편집하거나 복사

* 콘텐츠 범위
  * DOM 트리의 각 노드(텍스트 노드)의 시작 위치와 끝 위치의 범위
    * 이를 영역 포인트라고 함
  * 즉, 영역 포인트(시작, 끝) 사이의 콘텐츠(텍스트)를 편집, 복사
